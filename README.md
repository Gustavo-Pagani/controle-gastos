# controle-gastos
Projeto de estudo para desenvolvimento de um sistema de controle de gastos pessoais utilizando Python e MySQL, com cadastro e consulta de receitas e despesas.

## Objetivo

Este projeto foi criado com foco em estudo prático para aprender:

- conexão do Python com MySQL
- organização de projeto em arquivos e pastas
- criação de interface gráfica com Tkinter
- implementação de CRUD completo
- boas práticas básicas para projetos iniciantes

## Tecnologias utilizadas

- Python
- Tkinter
- MySQL
- mysql-connector-python
